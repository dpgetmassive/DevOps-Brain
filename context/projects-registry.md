## Project Status Values

- **Planning**: Initial setup, requirements gathering
- **In Progress**: Active development
- **Review**: Code review, compliance validation
- **Testing**: QA, test validation
- **Done**: Deployed, documented, closed
- **On Hold**: Paused, blocked, or deprioritized

## Compliance Status Values

- **Not Started**: Compliance checklist not yet reviewed
- **In Progress**: Compliance validation ongoing
- **Compliant**: All compliance requirements met
- **Blocked**: Compliance issues preventing progress
